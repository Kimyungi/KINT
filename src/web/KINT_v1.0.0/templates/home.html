KINT API

신어 감지, 예문추출, 감성분석 Model과 신어 DB를 통합 Open API로 제공합니다.

신어 감지 Model

약 1300만개의 분야별 데이터를 수집하여 soynlp 기반으로 문장이나 문서를 용어 단위로 전처리한 후 KINT의 11개의 통계변수에 따라 신어 감지

감성 분석 Model

Point Mutual Information(PMI) 기술을 기반으로 신어에 대한 감성을 분석하고 용어간의 연관도를 분석하여 연관 키워드를 추출

예문 추출 Model

Bidirectional GRU와 N-gram을 기반으로 신어의 예문을 추출하여 신어가 사용되는 상황과 뜻 파악

신어 통합 DB

KINT의 3가지 Model를 통해 정제된 신어 데이터와 신어에 대한 통계분석 결과, 감정분석 결과, 연관 키워드 및 예문추출 결과를 통합하여 제공

KINT Search

KINT API를 기반으로 한 다양한 분석 결과를 웹 기반 검색 서비스로 제공합니다.

    Example sentence
    Sentiment analysis
    Statistical analysis
    Related keywords

KINT Search는 KINT Open API의 Model들을 통해 도출된 다양한 분석 결과를 시각화하여 보여줍니다. 예문을 통해 인터넷 신어들어 어떤 문맥에서 쓰는지 파악하고 감성분석 결과를 통해 해당 검색어가 문맥에서 얼마나 긍정적으로 그리고 부정적으로 쓰였는지 확인할 수 있습니다. 시계열 분석을 통해 시간별 빈도 분석 결과와 분야열 사용 비율을 그래프로 시각화하여 해당 단어가 어떤 분야에서 주로 쓰이고 언제부터 쓰였는지 쉽게 알 수 있습니다. 또한 연관 키워드를 최대 5개까지 제공합니다. KINT Search는 KINT API를 사용하지 않아도 누구나 인터넷 신어에 대해 쉽게 이해할 수 있도록 개발된 서비스입니다. 마지막으로, 이달의 KINT를 통해 인터넷 신어 트랜드를 한눈에 확인하세요!
