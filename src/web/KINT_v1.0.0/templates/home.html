KINT Search는 KINT API의 결과들을 통해 다양한 분석 결과를 시각화하여 보여줍니다. 예문을 통해 인터넷 신어들어 어떤 문맥에서 쓰는지 파악하고 감성분석 결과를 통해 해당 검색어가 문맥에서 얼마나 긍정적으로, 부정적으로 쓰였는지 확인할 수 있습니다. 시계열 분석을 통해 시간별 빈도 분석 결과와 분야열 사용 비율을 그래프로 시각화하여 해당 단어가 어떤 분야에서 주로 쓰이고 언제부터 쓰였는지 쉽게 알 수 있습니다. 또한 연관 키워드를 최대 5개까지 제공합니다. 위의 분석 결과를 통해 KINT API를 사용하지 않아도 누구나 인터넷 신어에 대해 쉽게 이해할 수 있도록 했습니다. 마지막으로, 이달의 KINT를 통해 인터넷 신어 트랜드를 한눈에 확인해세요!
